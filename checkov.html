CKV_AWS_16

RDS Database Password Hardcoded
CKV_AWS_17
Database passwords should never be hardcoded in Terraform files. Use AWS Secrets Manager or SSM Parameter Store.
Resource: aws_db_instance.database

How to Fix:

# Create a random password resource "random_password" "db_password" { length = 32 special = true } # Store in AWS Secrets Manager resource "aws_secretsmanager_secret" "db_password" { name = "myapp-db-password" } resource "aws_secretsmanager_secret_version" "db_password" { secret_id = aws_secretsmanager_secret.db_password.id secret_string = random_password.db_password.result } # Use in RDS instance resource "aws_db_instance" "database" { # ... existing configuration ... password = random_password.db_password.result }

Security Group Allows Unrestricted Ingress
CKV_AWS_23, CKV_AWS_260
Security group allows ingress from 0.0.0.0/0 on all ports, exposing resources to potential attacks from anywhere on the internet.
Resource: aws_security_group.wide_open

How to Fix:

resource "aws_security_group" "web" { name = "web-security-group" description = "Security group for web servers" vpc_id = aws_vpc.main.id # Allow HTTP from specific sources only ingress { description = "HTTP from load balancer" from_port = 80 to_port = 80 protocol = "tcp" cidr_blocks = ["10.0.0.0/16"] # Replace with your VPC CIDR } # Allow HTTPS from CloudFront ingress { description = "HTTPS from CloudFront" from_port = 443 to_port = 443 protocol = "tcp" cidr_blocks = ["10.0.0.0/16"] # Or use prefix list for CloudFront } # Restrict egress to specific destinations egress { description = "HTTPS to AWS services" from_port = 443 to_port = 443 protocol = "tcp" cidr_blocks = ["0.0.0.0/0"] # Consider restricting further } }
IAM Policy Uses Wildcard Actions
CKV_AWS_63
IAM policies should follow the principle of least privilege and avoid using wildcard (*) actions.
Resource: aws_iam_policy.admin

How to Fix:

resource "aws_iam_policy" "app_policy" { name = "app-specific-policy" description = "Policy with specific permissions for application" policy = jsonencode({ Version = "2012-10-17" Statement = [ { Effect = "Allow" Action = [ "s3:GetObject", "s3:PutObject", "s3:DeleteObject" ] Resource = [ "${aws_s3_bucket.data.arn}/*" ] }, { Effect = "Allow" Action = [ "rds:DescribeDBInstances", "rds:DescribeDBSnapshots" ] Resource = [ aws_db_instance.database.arn ] } ] }) }
